# End-to-End Sales Performance Analysis (SQL + Power BI)

## Project Overview
This project performs an end-to-end sales performance analysis using SQL and Power BI.

The objective is to transform raw transactional data into structured insights, ensure data accuracy, and support business decision-making through monitoring and evaluation.

---

## Tools & Technologies
- SQL (Data Query, Aggregation, Filtering)
- Power BI (Data Visualization & Dashboard)
- Microsoft Excel (Data Preparation)

---

## Key Insights
- Identified monthly sales trends and performance patterns  
- Detected fluctuations and anomalies in sales data  
- Highlighted top-performing and underperforming products  
- Provided structured insights for business evaluation  

---

## Quality & Data Validation
- Performed data validation to ensure accuracy and consistency  
- Checked for missing, duplicate, or inconsistent data  
- Ensured reliable data before generating reports  
- Maintained structured and traceable data processing  

---

## SQL Analysis
- Used aggregation functions (SUM, AVG) for KPI calculation  
- Applied filtering and grouping for performance segmentation  
- Created structured queries for reporting and monitoring  

---

## Dashboard (Power BI)
This project includes a dashboard to visualize:
- Monthly performance trends  
- Product performance comparison  
- Sales distribution  

---

## Dataset
Dataset used for analysis contains transactional sales data including product, date, and revenue.

---

## Key Takeaway
This project demonstrates the ability to:
- Analyze data using SQL  
- Ensure data quality and accuracy  
- Identify business insights  
- Support decision-making through structured reporting
