# End-to-End Sales Performance Analysis (SQL + Power BI)

## Project Overview
This project performs an end-to-end sales performance analysis using SQL and Power BI.

This project simulates real-world sales data analysis to support business monitoring and decision-making. The objective is to transform raw transactional data into structured insights, ensure data accuracy, and provide reliable information for performance evaluation.

---

## Tools & Technologies
- SQL (Data Query, Aggregation, Filtering)
- Power BI (Data Visualization & Dashboard)
- Microsoft Excel (Data Preparation)

---

## Key Metrics
- Revenue trend
- Monthly performance (MoM)
- Product performance
- Sales distribution

---

## Key Insights
- Monthly sales show fluctuating patterns with certain periods indicating performance drops  
- Several products consistently outperform others and contribute the highest revenue  
- Some categories indicate declining trends
